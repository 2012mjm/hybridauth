# HybridAuth 3.0.0-dev branch

This branch contains work in progress toward the next HybridAuth 3 release and subject to change at any time.

Bundled Providers
-----------------
* Google (done)
* Facebook (partly done)
* Twitter (partly done)
* Yahoo
* Windows (partly done)
* LinkedIn
* OpenID(s) (partly done)

Requirements
------------
* PHP 5.3
* PHP/libcurl enabled
